# java-filmorate
В проекте реализованы:

- Добавление отзывов
- Режиссёры
- Лента событий пользователя
- Возможность удаления фильмов и пользователей
- Вывод самых популярных фильмов по жанру и году
- Рекомендации фильмов
- Вывод общих фильмов
- Поиск по названию и режиссёру
